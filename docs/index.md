# arc-exporter

Safely migrate Arc Browser profiles into another browser — without losing
bookmarks, passwords, cards, cookies, extensions, or open tabs.

This site is the user-facing documentation. The source code, issues, and changelog
live on [GitHub](https://github.com/mhadifilms/arc-exporter).

## Where to start

- [**Install**](install.md) — one-liner for every OS.
- [**Quick start**](quickstart.md) — five commands and you're done.
- [**Per-browser guides**](targets/index.md) — what to expect when migrating to
  each supported target.
- [**Troubleshooting**](troubleshooting.md) — `doctor` errors, Keychain prompts,
  Full Disk Access, and more.
- [**Security & privacy**](security.md) — what touches disk, what doesn't, and
  why.
